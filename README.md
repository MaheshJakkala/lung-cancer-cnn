# Lung Cancer CT Scan Classification using CNN

## Overview
This project implements an end-to-end deep learning pipeline for
classifying lung CT scan images into Normal, Benign, and Malignant
categories using Convolutional Neural Networks (CNNs).

The goal is to demonstrate applied computer vision, medical image
preprocessing, and model evaluation techniques.

## Dataset
IQ-OTHNCCD Lung Cancer CT Scan Dataset (Kaggle)

## Methodology
- Exploratory Data Analysis (EDA)
- Image preprocessing & normalization
- CNN architecture with Keras
- Model training and evaluation

## Results
- Test Accuracy: **99.55%**
- High precision and recall across all classes
- Confusion matrix and training curves included

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy, Pandas
- Matplotlib, Seaborn
- Scikit-learn
